## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sniffets'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sniffets

Run the generator:

    $ rails generate sniffets:install

And migrate:

    $ rake db:migrate
